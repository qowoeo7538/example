### 1.2API
- Class<?> defineClass(String, byte[], int, int) :将一个 byte 数组转换为 Class 类的实例。这个方法被声明为final的。
- Class<?> findClass(String) :使用指定的二进制名称查找类。
- Class<?> loadClass(String) :使用指定的二进制名称来加载类。
- void resolveClass(Class<?>) :链接指定的类。
